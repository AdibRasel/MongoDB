MongoDB Setup

MongoDB Setup Localhost

সর্বপ্রথম MongoDB ওয়েব সাইট থেকে MongoDB- Enterprise Advanced অথবা MongoDB Community Server সফটওয়্যার ডাউনলোড করতে হবে।
আমরা MongoDB Community Server দিয়ে MongoDB সেটআপ করবো। Download MongoDB Community Server ডাউনলোড করতে এইখানে ক্লিক করুন।

তারপর MongoDB Compass ডাউনলোড করতে হবে। MongoDB Compass ডাউনলোড করতে এইখানে ক্লিক করুন।

MongoDB Community Server থেকে সার্বার সেটআপ

MongoDB Community Server ইন্সট্রল হয়ে গেলে লোকাল ড্রাইভে C:\Program Files\MongoDB\Server\5.0\bin এই পাথ(path) এ ছলে আসতে হবে। পাথ(path) টি হলোঃ- C:\ → Program Files → MongoDB → Server → 5.0 → binএই পোল্ডার এ চলে আসতে হবে। এই পাথ এ আসার পর লোকাল ড্রাইভের লিংকে CMD লিখে এন্টার দিলে কমান্ড প্রোমট চলে আসবে, সেখানে টাইপ প্রথমে টাইপ করতে হবে mongod লিখে এন্টার দিতে হবে, তারপরে mongo লিখে এন্টার দিতে হবে, তাহলে নতুন করে একটা সার্ভার তৈরি হয়ে যাবে।

mongod লিখে এন্টার দিতে হবে

তারপর mongo লিখে এন্টার দিতে হবে।

mongo লিখে এন্টার দেওয়ার পরে একটা সার্ভার তৈরি হয়ে যাবে। সার্ভার Address হচ্ছেঃ- mongodb://127.0.0.1:27017/

সার্বার আইপি দিয়ে কানেক্ট করা।

mongodb://127.0.0.1:27017/ সার্ভারের এই Address টি MongoDB Compass এর সাথে কানেক্ট করে ডাটাবেস এর সকল কাজ করতে হবে

কানেক্ট হয়ে গেলে সকল কাজ করা যাবে।
